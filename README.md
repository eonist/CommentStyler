# CommentStyler

A CommentStyler app

### Features:

- Converts JS-style code blocks to Obj-c style code blocks

<img width="407" alt="img" src="convert.gif?raw=true">

### Example:

```swift
/**
 * Comment
 */

 To:

 ///
 /// Comment
 ///
```

### Instructions:

1. select and copy the code you want to format
2. click the convert-comment-block button
3. paste the code
